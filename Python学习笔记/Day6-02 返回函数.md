返回函数顾名思义（外层）函数的返回值是一个函数而不是值，这样做的好处是，既然返回的是一个函数，那么就可以按需定制。返回一个我们需要的函数。
比如下面的例子里面，根据需要，通过设置外层函数的参数，create_calculator返回了两个函数add_func和sub_func

```python

# 需求：根据传入的运算类型（加/减），返回对应的运算函数
def create_calculator(operation):
    # 定义加法函数
    def add(a, b):
        return a + b
    
    # 定义减法函数
    def subtract(a, b):
        return a - b
    
    # 根据参数返回不同的函数
    if operation == "add":
        return add
    elif operation == "subtract":
        return subtract

# 生成加法函数
add_func = create_calculator("add")
print(add_func(5, 3))  # 输出：8

# 生成减法函数
sub_func = create_calculator("subtract")
print(sub_func(5, 3))  # 输出：2
```
