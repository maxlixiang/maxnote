partial函数，我感觉翻译成部分函数也不错

`partial`函数的**第一个参数和返回值都是函数**，它将传入的函数处理成一个新的函数返回

```python
import functools

int2 = functools.partial(int, base=2)
int8 = functools.partial(int, base=8)
int16 = functools.partial(int, base=16)

print(int('1001'))    # 1001

print(int2('1001'))   # 9
print(int8('1001'))   # 513
print(int16('1001'))  # 4097
```
