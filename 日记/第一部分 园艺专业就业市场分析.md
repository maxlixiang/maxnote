
# 背景

目前大帅帅马上进入大二下学期，并且计划考研。为了让咱的大学生活更充实，不浪费宝贵的学习时间，毕业后好找工作，所以在这做一个大学期间的学习指导。从就业市场的情况来看，单一技能的人才并不稀缺，如果大学期间只是学习专业技能，那么到了毕业找工作的时候很可能因为能力不够突出而找不到心仪的工作。正确的做法是在保证专业技能的同时，附加一门额外的技能，**让自己变成复合型人才**。以园艺学专业的就业方向举例，如果毕业后想从事景观与园林工程方面的工作，那么除了学习本专业知识以外，附加学习CAD、Photoshop等制图软件，使自己具备为工程和产品制图的能力，就能极大的提到自己的竞争优势。再比如如果想从事花卉与观赏园艺方面的工作，那么提前了解直播电商的相关知识也能为毕业后的就业增加议价空间。

# 个人情况分析

孩子对AI和编程比较感兴趣，在初中也学过一些python编程，有一定的基础。再加上数学成绩也不错，**最现实、最适合园艺 + AI**的就业路线：园艺本专业扎实 + 编程 / AI 当工具 + 往**智慧农业、数字园艺方向**走。这是现在最缺人、就业面最宽的组合。哪怕以后不从事园艺相关工作，单独拿编程/AI的能力找工作也不是没有可能的。AI这几年的发展非常迅速，不管是国内还是国外，各大科技公司都汇聚了顶尖的人才进行研发。AI的发展会深刻影响人类生活，极大提高生产效率，有可能为人类开启新一轮的工业革命。所以学一些AI和编程类的知识，在就业的时候非常具有竞争优势。

# 学习大方向：别丢园艺，也别死磕计算机

园艺专业 + 会编程 / AI = 稀缺复合型人才，纯计算机的人不懂植物，纯园艺的人不会代码，你刚好卡在中间。
**推荐路线：**
- 主修：园艺学（学硕）
- 辅修 / 自学：**Python + 数据处理 + 农业 AI 基础**
- 未来方向：智慧农业、设施园艺、数字植物表型、农业大数据、园艺智能装备
## 大二：打基础（最关键一年）
1）稳住园艺本专业
重点学好：
植物生理学、园艺植物育种、设施园艺、土壤学、试验设计与统计
保持GPA 3.0+/4.0，考研非常看本科成绩
2）计算机 / AI 只学最有用的
只学 Python，别一上来搞 C++/Java
课程推荐（B 站 / 慕课都行）：
Python 基础
数据处理：Pandas、NumPy
数据可视化：Matplotlib、Seaborn
目标：
能用代码处理实验数据、画图表、做简单分析
3）要不要辅修计算机？
不建议辅修，太占时间，还影响考研。更划算的是：**自学核心技能 + 蹭课（数据库、机器学习导论）**

## 大三：往 “园艺 + AI” 靠拢
1）进实验室
优先选这些方向的课题组：
设施园艺 / 智能温室，植物表型组学，农业大数据、遥感与园艺，水肥一体化、智能监测
**会点 Python，老师会非常喜欢你。**
2）进阶技能
机器学习基础（不用深）
用 Python 做：作物生长模型，图像识别（叶片病害、果实分级）
工具：
Scikit-learn
简单深度学习：PyTorch 入门
3）开始准备考研
英语：每天背单词、做阅读
政治：大三下再开始
专业课：跟着本专业课堂走
## 大四：考研 + 强化技能
上半年：全力考研
考研结束后：
做一个园艺 + AI 小项目（比如：识别植物叶片病害）
写进简历，复试、找工作都巨加分
## 未来就业 / 发展？
你这种背景能去的地方：
农业科技公司（智慧农业、AI 农业）
种业公司（数据分析、育种辅助）
设施农业企业（智能温室、环境控制）
公务员 / 事业单位（农业农村局、农技推广）
读博：往农业信息学、数字园艺走，很容易发文章

- 大三
进园艺 + 智能 / 数据相关实验室
学机器学习基础

- 大四
考研
