## 总原则

- **主线**：园艺专业课稳住 + 为考研打基础
- **副线**：Python 学到能帮你做实验、写论文、复试加分
- **核心要求**：不搞花里胡哨，只学最有用的

---

## 一、园艺专业课

### 核心目标

GPA 越高越好（复试非常加分），争取专业排名前 30% 以内（越靠前越好）

### 重点科目

1. 植物生理学（学科代码 414，考研极核心）
2. 园艺植物栽培学 / 育种学
3. 试验统计与数据分析（和编程直接联动）
4. 设施园艺、田间试验设计

### 学习方法

1. 每节课做笔记 + 课后当天过一遍
2. 期末提前 3 周开始系统复习

---

## 二、Python & AI（定向学习，不盲目拓展）

### 阶段目标（大二下）

1. 会用 Python 处理园艺实验数据
2. 会画图、会做统计、会写进课程论文
3. 开始留意**智慧农业、设施园艺、表型组学**相关老师

### 学习路线（按顺序推进）

表格

|学习模块|核心内容|
|---|---|
|Python 基础|变量、循环、条件、列表、字典、文件读取（2～3 周搞定，能写简单脚本即可）|
|数据处理三件套|NumPy（数值计算）、Pandas（处理 Excel / 实验数据）、Matplotlib（绘制生长曲线 / 柱状图 / 折线图）|
|统计与试验分析|均值、标准差、显著性、相关分析（能用代码复现试验统计结果）|
|可选拓展|Scikit-learn 基础，完成植物生长预测 / 病害分类简单模型（机器学习入门，不着急）|

### 时间安排（每周）

- 周一～周五：每天 40～60 分钟 学 Python
- 周末：一次 2 小时 集中练手
- 总投入：每周 5 小时以内（不影响考研主线）

---

## 三、考研准备（大二下仅做 3 件事）

### 1. 英语（每日必做）

- 背单词：15～20 分钟 / 天
- 长难句 / 阅读：每周 2～3 篇
- 目标：大三直接刷真题不费劲

### 2. 确定考研方向（大二下必须明确）

- 推荐方向：园艺学硕（设施园艺 / 智慧农业 / 植物表型 / 农业信息学）

### 3. 基础铺垫（考研核心内容预热）

|模块|具体要求|
|---|---|
|数学（农）|提前看 314 考纲，熟悉高数 / 线代 / 概率农科考点；教材用《高等数学》（同济）、《线性代数》、《概率论与数理统计》|
|专业课 414|吃透《植物生理学》《生物化学》教材，做思维导图；大二结束前实现 414 核心知识点无盲区|
|信息收集|华农园艺林学学院：查导师、招生目录、复试细则；浙大农业与生物技术学院：查招生名额、推免比例、复试要求|

## 四、实验室相关（大二下核心任务）

### 核心意义

1. **能力与认知**：落地课本知识，掌握实验操作、仪器使用、数据处理等核心技能，建立科研思维
2. **升学 / 就业**：科研经历是保研、考研复试、申请名校的核心加分项；企业也看重实践能力
3. **方向确认**：提前体验真实科研，判断是否适合读研、明确细分方向，避免后期走弯路

**千万不要只进行考研学习，我们的目标是考研，分数达标只是基础，导师愿不愿意要才是核心。** 复试的时候，老师会问：你做过什么实验？会用什么仪器？懂不懂 PCR、组培、生理指标测定？有没有进过实验室？

>同样考 350 分，有科研和没科研，差距天壤之别
只有分数，老师会觉得你是 “考试型选手”，能不能做科研不确定。老师心里会直接打鼓：这学生只会背书，来了我还得从头教，太费时间。
分数不错 + 1–2 年实验室经历，老师会觉得： 这孩子踏实、肯干、上手快、能干活。直接愿意收。

# 过来人的一点感悟

可能是年纪大了，这两年总喜欢回想以前的事儿🤡

我上大学那会，一晃都过去二十年了。那会儿没人教我该怎么规划大学生活，互联网也不发达，所有信息都只能靠身边人口口相传，所以我从头到尾都没给自己定过什么主线目标，就跟着舍友一起上课、泡图书馆，日子过得轻松快乐，最后也顺顺利利毕业了。可真到了毕业找工作，我还是陷进了好几个月的迷茫期。反观身边不少同学，早早便有了自己的打算，还有几个甚至没上大学前，就把工作、考研的路规划好了，毕业之后有人顺利就业，有人安心读研，那股笃定劲儿，真让我羡慕。

常常想，要是能重新读一遍大学，我肯定能做得更好，可人生哪有重来的机会，过去的时光终究是过去了。其实日子都是一样过，打游戏是一天，沉下心学习也是一天。追求躺平的生活固然轻松，可真要走到人生回头看的那一刻，又有几个人能做到问心无愧呢？**你每多学的一个知识点，每多背的一个单词，都会成为你未来前行的资本。**

这话当年老师也常跟我说，那时候只觉得是老生常谈的心灵鸡汤，可人到不同的人生阶段，对同一句话总会有不一样的感悟，如今才懂，老师当年是真的没骗我。可惜那时候的我没把这些话放在心上，更没将这份道理转化成往前走的动力，现在想来总有些遗憾。

在你身上，我总能看到几分当年自己的影子，也正因如此，才更希望你能好好珍惜当下的时光，踏踏实实走好每一步，在求学的道路上能走得更稳、更远。
